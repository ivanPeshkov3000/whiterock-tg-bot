# White Rock Telegram Bot

Node.js бот для автоматического поиска новостей про Белую Скалу (Крым)
через RSS, обработки с помощью AI и публикации в Telegram.

## Запуск

1. npm install
2. Создайте .env по примеру .env.example
3. npm start

## Команды

- npm start — запуск бота
- npm dev — запуск в watch режиме
- npm migrate — миграции Drizzle
